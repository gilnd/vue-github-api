# Vue.js + Github Api Showcase

Simple Vuejs + Github API showcase! :) 

For more detail read the [Documentation](https://github.com/gilnd/vue-github-api/blob/main/docs/doc.md).

### Project setup
```
npm install
```

### Run Project
```
npm run serve
```

### Run Tests
```
npm run test
```
